This is a role-playing system.
It is written by Harley Dutton and tested by Niel Bernheim.

Required materials are six-sided dice, paper, and pencils.
A storyteller running this system should have experience and foresight.
This system works well for novice role-players.
Setup and character creation should take less than an hour.

This system can accomodate any character or setting.
There are no canonical species, lore, setting, or special powers.

It should be noted that sometimes the guidelines don't really work.
Add to, modify, or ignore them when this happens.
